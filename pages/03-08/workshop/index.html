HTML Layout Essentials

You already know how to construct a basic HTML page, and how to style the HTML elements using CSS. Now, let’s take a look at how different kinds of elements display in a page layout.

1. Text and Elements

Text

Strings of text always appear in an inline flow. That means that every word will appear to the right of the one before it until there is no more space on the line, at which point, the string of text will start again at left side of the line below. Left to right, top to bottom. This is the flow of inline content.

But raw text alone doesn’t make for a great editorial layout. A good layout, like any web page, needs:

  1. Content to be able to be assembled in a variety of layout orientations, say for example, a grid.
  2. Content to be visual, interactive, and functional.

This is where elements come into play. Today, we are going to focus on how different elements let us layout an editorial web page in different ways.

Elements

An element is like a container you can put content in. While it’s not possible to give raw text different layout structures, styles, and interactive properties, it's possible to do this with elements that contain text. Moreover, elements can also contain embeddable media and even other elements. In html, an element is anything wrapped in an opening and closing tag.

Here are a few examples of elements:

  • <span>this is a string of text</span>
  • <p>this is a string of text</p>
  • <div>this is a string of text</div>
  • <div>
    • <p>this string of text is a header</p>
    • <p>this string of text is a paragraph.</p>
    </div>
  • <li>this is a string of text</li>

3. Aligning Inline & Block Elements

Say you want to align the content on your page. By default, all content is left-aligned. Beyond that...

For inline content

To align an inline element: you can give its parent container an attibute that aligns everything inside the parent left, center, or right.

For block elements

To align a block element: you can give the element itself an attribute that tells it how it should align itself on the page. One common objective is centering a block element. This can be done by setting its left and right margins to "auto" (e.g. margin-left: auto; margin-right: auto;)

Example

The example below shows one sentence. The sentence is contained in a block, div element. (Because it is a block element, I could set a width and height for it). I gave it the style attribute of margin-left: auto; margin-right: auto;. As a result, it centers itself in its parent.

What if I wanted to the raw text inside the divto be text-aligned center? In that case, you can think of the div as a parent element for the inline text content. I gave the div the style attribute of text-align: center;. Now, the div is both centering itself in its own parent, and it is centering all inline text inside it.

Floating elements

There is another way to align block and inline elements to the left or right. You can give the element the style attribute of float: left or float: right. This aligns it to the left or right side within its parent element. Giving an element the float attribute also does a couple other things:

  • While it will keep its original vertical position on the page, it will force itself to the right or left side of its own container, no matter what other elements, inline or block, are next to or below it.
  • While it will not overlap with sibling inline elements, it will overlap with sibling block elements.
