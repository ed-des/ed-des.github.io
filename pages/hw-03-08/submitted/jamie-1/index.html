HTML Layout Essentials

This page is a tutorial of how to use inline elements and block elements in HTML and CSS as tools for laying out text and webpages. The tasks go step by step and build up into more complex layouts using inline and block elements as basic structures. Made by Jamie Chen in Dylan Grief’s Editorial for Screen Class, RISD Spring 2016.

Task 1: Example of grouping Element

This is a section of stuff.An element is like a container you can put content in. While it’s not possible to give raw text different layout structures, styles, and interactive properties, it's possible to do this with elements that contain text. Moreover, elements can also contain embeddable media and even other elements. In html, an element is anything wrapped in an opening and closing tag.

Task 5:Example of inline-block elements

You can make an element that behaves like both an inline and block element.In CSS, all you have to do is add the display: inline-block; attribute to the element. An inline-block element will be built as if it were a block element, but it will flow on the page as if it were an inline element.
