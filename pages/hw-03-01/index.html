< Editorial Design for Screen

Assignment from 03-01

Look for five inspiring quotes, varying from 500 – 1000 characters long. Design each one as its own web page.

Each web page should be solved in its own folder, comprised of its own HTML and CSS file. Label each folder as firstname-1, firstname-2, firstname-3, etc. Label the html file in each folder index.html.

Each should include a header, the quote itself, an attribute, and any stylistic elements you choose. Think about: balance, legibility, dynamism, softness, loudness, and contrast... Play with: font size, scale, leading, alignments, contrast, margins...

For quotes, check out Maria Papova’s Brain Pickings. (Posts usually include quotes from writers, scientists, artists, and philosophers.) Here is a characters count tool if you need one.

Upload all five folders to our shared google drive folder “Assignments” by 11:00AM next Tuesday.
